 Arduino LED Blink Using Digital Output 

 Overview
This project demonstrates the basic operation of an Arduino Uno by blinking the built-in LED connected to digital pin 13. It is one of the fundamental beginner projects in embedded systems and Arduino programming, helping users understand digital output control, program structure, and timing using delays.

 Objectives
* Learn the basic structure of an Arduino program.
* Configure a digital pin as an output.
* Control an LED using digital signals.
* Understand the use of the `setup()` and `loop()` functions.
* Demonstrate timing control using the `delay()` function.

Components Required
* Arduino Uno
* USB Cable
* Arduino IDE
* Computer/Laptop

Note: No external LED or resistor is required since the Arduino Uno has a built-in LED connected to digital pin 13.

 Circuit Description
The project uses the built-in LED available on the Arduino Uno board. Since the LED is internally connected to digital pin 13, no external wiring is necessary.

Working Principle
When the Arduino starts, pin 13 is configured as an output. The program continuously turns the LED ON for one second and OFF for one second. This cycle repeats indefinitely, creating a blinking effect.

 Program Flow
1. Start the program.
2. Initialize pin 13 as an output.
3. Turn the LED ON.
4. Wait for one second.
5. Turn the LED OFF.
6. Wait for one second.
7. Repeat the process continuously.
